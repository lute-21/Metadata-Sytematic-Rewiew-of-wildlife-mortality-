This repository contains a metadata file (`Metadados_luiza_backup.xlsx`) compiling bibliographic and screening information for articles retrieved during a systematic literature review.

## File Description

Each row corresponds to a single article identified in scientific databases. The file was designed to support study screening, duplicate checking, data extraction, and transparency throughout the review process.

## Main Columns

* **Authors / Article Title / Source Title**: Bibliographic identification of each study.
* **Publication Year, Volume, Issue, Pages, DOI, UT (Unique ID)**: Publication and indexing details.
* **Plataforma**: Database or search platform where the article was retrieved.
* **Keyword_language / Subject / Grupo**: Thematic and categorical classification used for analysis.
* **Duplicata**: Identification of duplicate records across databases.
* **Abstract / Title (Reviewer-specific columns)**: Versions used during screening.
* **Andreas_Article / Luiza_Article**: Inclusion or exclusion assessment performed independently by two reviewers.
* **Obs**: Additional notes relevant to screening or data quality.

## Purpose

This metadata file ensures organization, traceability, and reproducibility of the systematic review. It enables transparent reporting of search results, independent screening by reviewers, and subsequent descriptive and qualitative analyses.

## Usage

The file can be used for:

* Documenting the study selection process
* Supporting PRISMA flow diagrams
* Conducting descriptive statistics and thematic synthesis
